# Застосунок класифікації емоцій

## Короткий опис

Даний застосунок розроблено в межах кваліфікаційної роботи студента групи ШІ-41 Дуди Артема Сергійовича

Репозиторій містить як ноутбуки з експериментами, так і код для тренування та оптимізації гіперпараметрів моделі архітектури CRNN та донавчання моделі HuBERT. 

Проект робить класифікацію 6 класів емоцій: 
 - Злість 
 - Радість 
 - Сум
 - Нейтральність 
 - Страх
 - Здивування

## Інструкція запуску 

Потрібно встановити [UV](https://docs.astral.sh/uv/getting-started/installation/) та переконатися, що інструмент успішно встановлено (перезапустити термінал і прописати ```uv --version```);

Перейти у кореневу диркеторію склонованого проекту і тільки тоді синхронізувати залежності.
```bash
uv sync
```
### Запуск веб-інтерфейсу

Щоб запустити веб-інтерфейс за посиланням http://localhost:8000 Перейти у кореневу диркеторію склонованого проекту і тільки тоді запустити команду:
```bash
uv run manage.py runserver
```

### Запуск експериментів та оптимізації моделей

У Jupyter ноутбуках в локальній кореневій директорії проекту ```notebooks/``` знаходяться приклади запуску конвеєрів попередньої обробки даних послідовностями команд для проведення експериментів, які можна взяти за приклад і виконувати.

NOTE: Модель HuBERT відсутня у репозиторії, оскільки важить ~300MB. Щоб її отримати, потрібно виконати кроки експерименту 3, після чого знайти її у папці ```temp``` та перемістити у ```trained_models```.
